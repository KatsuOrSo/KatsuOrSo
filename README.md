### Good to see you!  
I`m a system administrator / operator and full-stack developer. Currently, I'm still a student, but at the moment I'm already working part-time for and with some big, as well as small companies. My goal is, with my (more than) "hobby", to be self-employed later and always create new and innovative products.

I specialize in cloud platforms and applications, with a focus on remaining as platform-agnostic as possible, so that I can develop, deploy or manage programs and systems across vendors, systems and existing capabilities. Be it an own program or huge database clusters on Docker or Kubernetes basis.
In my work, I focus a lot on high availability architectures and solutions to ensure zero downtime at lowest cost and highest security.

For this, I deal, in addition to a lot of automation and cloud stuff, with threat detection systems on network and system basis, attack tactics and protection options. (Be it firewalls, pro-active systems, virus scanners on server / cloud and much more).
